# Blackey — Explore Africa Like Never Before

Blackey is Africa's travel super-platform, bringing flights, hotels, safaris, and more together in one place. Built to serve travellers across all 54 African countries.

## Tech Stack

- Next.js 15 + TypeScript
- Tailwind CSS 4
- Framer Motion
- Playfair Display + DM Sans (via `next/font`)

## Features

- **Sticky Navbar** — 7 service tabs, currency/language selector, sign in/register
- **Hero + Search Widget** — Full-viewport African landscape with frosted glass tabbed search
- **Popular Destinations** — Horizontal scroll of 10 African cities with pricing
- **Hot Deals** — 3-column grid with countdown timers, discount badges, and Book Now CTA
- **Explore by Service** — 7 icon cards with gold hover fill animation
- **Why Blackey** — Africa-First, Local Payments, Secure Bookings, 24/7 Support
- **Featured Experiences** — Parallax editorial cards (Serengeti, Zanzibar, Cape Town)
- **Footer** — Navigation, payment logos, social icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.
