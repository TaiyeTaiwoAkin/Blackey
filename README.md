# Blackey — Explore Africa Like Never Before

Blackey is Africa's travel super-platform, bringing flights, hotels, safaris, and more together in one place. Built to serve travelers across all 54 African countries.

## Features

- **Hero Section** — Full-viewport African landscape background with parallax scroll and dark gradient overlay
- **7-Tab Search Widget** — Frosted glass search interface with tabs for:
  - Flights
  - Hotels
  - Car Rentals
  - Packages
  - Activities
  - Cruises
  - Rentals
- **Popular Destinations** — Horizontally scrollable cards showcasing trending African cities
- **Limited Time Deals** — Featured travel deals with pricing and discounts
- **Explore by Category** — Quick-access grid for all travel categories
- **Featured Experiences** — Curated safari, beach, and city break adventures
- **Newsletter & App Download** — Email signup and mobile app links
- **Responsive Design** — Fully responsive across desktop, tablet, and mobile

## Tech Stack

- HTML5
- CSS3 (custom properties, backdrop-filter, CSS Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display, DM Sans)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/TaiyeTaiwoAkin/Blackey.git
   ```
2. Open `index.html` in your browser.

No build tools or dependencies required.
